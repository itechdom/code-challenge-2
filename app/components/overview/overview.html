Welcome everyone

The MWI Digital Standards Guide is here!

Why have a Digital Standards Guide?

Two big reasons:
  • To convey a consistent brand across all MWI digital properties
  • To make development faster and more efficient (remember, that’s not the same as magically complete—there’s still a lot of work to do to apply these elements and styles)

Principles we have followed in making style guide

We have followed these principles when we desinged the DSG. ITCSS, Atomic Design and BEM. You can read more about here

The folder structure

The main folder you should look at is the app/ folder. It contains docs and src folders. Src is all the Sass, HTML and JS files you will need (in case you want to copy any code). Docs contains all the files that are responsible for documenting Src.

The best way to find code

We tried to match the folder structure to the navigation in this site. For example, Atoms in the menu bar corresponds to app/src/views/atoms.

What browsers are supported?

On desktop we support the following:
  • Internet Explorer 8/9/10/11 (IE8 isn't fully supported by all styles, though it should function properly)
  • Firefox 32+
  • Chrome 37+
  • Safari 6.1+

What happens next?

This is just the beginning. Some sites have already been selected to undergo redesign. As sites undergo redesigns, additional elements that are needed will be designed and baked into the guide.
